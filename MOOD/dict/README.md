# 情感分析（基于字典）

- 基于大连理工大学“情感词汇本体"字典
- 引用：徐琳宏,林鸿飞,潘宇,任惠,陈建美.情感词汇本体的构造.情报学报,2008,27(2):180-185

## 实现
### 变量定义：

#### 乐
- PA 快乐
- PE 安心
#### 好
- PD 尊敬
- PH 赞扬
- PG 相信
- PB 喜爱
- PK 祝愿
#### 怒
- NA 愤怒
#### 哀
- NB 悲伤
- NJ 失望
- NH 内疚
- PF 思念
#### 惧
- NI 惊慌
- NC 恐惧
- NG 害羞
#### 恶
- NE 烦闷
- ND 憎恶
- NN 贬责
- NK 妒忌
- NL 怀疑
#### 惊
- PC 惊奇

### 流程：
使用AC自动机对句子进行匹配，将结果存入切片，之后按照一定算法（直球对比，只对比了强度，没管极性与词性，导致结果很可能不准确）对该句的情感进行计算。
